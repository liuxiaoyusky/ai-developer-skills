- Initial release of n8n-mcp-workflow skill
- Complete MCP server template with 4 core tools:
  - trigger_workflow - Trigger n8n workflows by ID or name
  - create_workflow - Create new n8n workflows programmatically
  - list_workflows - List all n8n workflows
  - get_workflow - Get specific workflow by ID
- Comprehensive n8n workflow structure documentation
- Common node types reference (Webhook, HTTP Request, Code, Set, IF)
- Connection patterns (Linear Flow, Conditional Branching, Multiple Outputs)
- 3 workflow creation patterns with complete code examples
- Top 5 common mistakes and solutions
- Best practices guide
- Quick reference checklist
- External resources and links to n8n community
