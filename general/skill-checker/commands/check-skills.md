# /check-skills

检查并显示所有本地skills和marketplace插件的更新状态。

## 使用方法

```bash
/check-skills
```

## 功能

1. 扫描本地skills目录
2. 读取已安装的marketplace插件
3. 检查每个marketplace是否有更新
4. 生成详细的状态报告

## 输出

显示：
- 本地skills列表（官方/自定义）
- Marketplace插件列表
- 可更新的marketplace及受影响插件
- 更新命令建议
